# Pipeline-only verification with radical transparency

No human reviews candidate scores before launch. Quality control is
structural instead: a tiered source policy (Tier 1: pamphlet, candidate's
own site/filings, voting records; Tier 2: endorsements, established news;
social media and opposition sites excluded), an adversarial refute pass
over every score, citations displayed on every claim, Evidence Level
shown honestly, and a report-an-error button whose fixes deploy via git
push in minutes.

Context that makes this defensible rather than reckless: this is a free
tool by one interested person who makes no accuracy claims, and the site
says so up front — built by AI, researched by AI, verify via the
citations. The alternative (human review of ~200+ candidates) did not
fit the 5-day window to the July 21, 2026 ballot drop, and we chose
transparency about the method over a review step we couldn't actually
afford. A future reader should not "fix" this by quietly adding accuracy
claims; the no-claims framing is the load-bearing part.
