# Spokane County Package

Status: ingested, not app-supported.

This package contains official Spokane County source pointers and raw current
election pages for the August 4, 2026 primary. It is not yet included in
`coverage.supported_counties` because the county still needs parsed
contests/measures, dossiers, scoring files, and district scoping before
address-matched voters can receive full county coverage.

Until then, Spokane voters should receive the statewide-only partial guide.
