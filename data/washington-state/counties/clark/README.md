# Clark County Package

Status: ingested, not app-supported.

This package contains official Clark County source pointers and the raw current
election page for the August 4, 2026 primary and special election. It is not yet
included in `coverage.supported_counties` because the county still needs parsed
contests/measures, dossiers, scoring files, and district scoping before
address-matched voters can receive full county coverage.

Until then, Clark voters should receive the statewide-only partial guide.
